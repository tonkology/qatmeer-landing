# Team Photo Instructions

## How to Replace Team Photos

1. **Photo Requirements:**
   - Format: JPG or PNG (JPG recommended for smaller file size)
   - Size: 300x300 pixels (square format)
   - Professional headshot style
   - Clear, well-lit photo
   - File size: Keep under 200KB for fast loading

2. **Replace These Files:**
   - `team-mohammed-tonkal.jpg` → Mohammed Tonkal's photo
   - `team-hassan-alharbi.jpg` → Hassan Alharbi's photo  
   - `team-albaraa-jaafar.jpg` → Albaraa Jaafar's photo
   - `team-muath-bamuqabil.jpg` → Muath Bamuqabil's photo
   - `team-taiba-salah.jpg` → Taiba Salah's photo

3. **How to Replace:**
   - Keep the exact same filename
   - Replace the placeholder file with the actual photo
   - Ensure the photo is cropped to square (1:1 aspect ratio)
   - Commit and push to git to deploy changes

4. **Tips for Best Results:**
   - Use consistent lighting and background style across all photos
   - Crop photos to show head and shoulders
   - Ensure faces are clearly visible and well-centered
   - Consider using a consistent background color or style

The website will automatically use these photos once you replace the placeholder files and deploy the changes.
